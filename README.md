# Slack Google Search
**It posts top 5 hits from google**

![](http://i.imgur.com/F70jBP2.png)

### Installation & Setup

Create and configure a [slash command](https://api.slack.com/slash-commands)

Add a `config.json` file containing your unique token provided by Slack

```json
{
  "token": "<your-token>"
} 
```

Run the following

`npm i`

`node app`
